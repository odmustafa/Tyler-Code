SecuGen WebAPI - Python

WebAPI has 3 requirements:
1. SecuGen fingerprint devices. (USB connected) Examples --Hardware--
2. SecuGen fingerprint driver for SecuGen fingerprint device. --Software--
3. WebAPI client installed. You can find these here. --Software--


WebAPI is operational using these tools:
1. Win10 (x86 or x64 architectures); Win11 (x64 architecture)
2. Modern Browser. Examples: Chrome, Edge, Opera, etc...
